## 功能特性

### 1. URL 解析
- 自动补全 `http://` 或 `https://` 前缀
- 解析协议、域名、子域名、路径、查询参数等
- 识别 IP 地址、异常端口、可疑 TLD

### 2. 黑名单检测
- 内置已知钓鱼域名关键词库
- 品牌仿冒检测（如假冒 Apple、Microsoft 等）
- 高风险顶级域名识别（.tk、.xyz、.top 等）

### 3. 风险评分（0-100分）
- **70+**：极度危险 ☠️
- **50-69**：高风险 🚨
- **30-49**：中等风险 ⚠️
- **15-29**：低风险 ⚡
- **0-14**：相对安全 ✅

### 4. 高危网站提醒
- 可视化风险等级徽章
- 圆形进度条显示风险评分
- 详细检测报告和风险说明
- 处理建议和警告信息

### 5. 其他功能
- 检测历史记录（本地存储）
- 响应式设计，支持移动端
- 深色主题 UI

直接在浏览器中打开 `index.html` 即可使用该工具。
